# getTangentLines
  
This Inkscape plugin allows to get the tangent line between two circles, making it relatively easy to recreate objects in the style of the Aphex Twin logo (font: xylem). This code was primarily generated by AI, with minor modifications made manually by me. 
  
Put getTangentLines folder into Inkscape extension directory.  
  
Ordinary...  
[windows] C:\Users\[yourUsername]\AppData\Roaming\inkscape\extensions  
[mac, linux] ~/.config/inkscape/extensions
